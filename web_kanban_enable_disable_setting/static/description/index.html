Web Kanban enable/disable settings

"Attribute Enable Setting in kanban view"

Functionality

This module is developed to give an attribute 'enable_setting' to the view kanban for hide setting.

You need to define an attribute on view kanban 'enable_setting=false' on your view kanban for hide the setting displaying when hovering over header of view kanban 
enable_setting='false'
